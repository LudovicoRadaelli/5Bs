Equazioni logaritmiche per sostituzione

/



Prima di tutto impariamo a riconoscere equazioni logaritmiche di questa tipologia.

In esse i logaritmi che figurano hanno tutti lo stesso argomento e sostituendo ogni logaritmo con una lettera qualsiasi otteniamo una equazione che sappiamo risolvere.

Facciamo alcuni esempi:


  • \( log^2\left(x\right) + 3\,log\left(x\right) - 2 = 0 \)

    infatti effettuando la sostituzione \[ \color{red}{}A\color{black}{} = \color{red}{}log\left(x\right) \] otteniamo un'equazione di secondo grado \[ \begin{align*} \color{red}{}log\color{black}{}^2\color{red}{}\left(x\right)\color{black}{} + 3\,&\color{red}{}log\left(x\right)\color{black}{} - 2 = 0 \\\\ &\Downarrow \\\\ \color{red}{}A\color{black}{}^2 + &3\color{red}{}A\color{black}{} - 2 = 0 \end{align*} \]


  • \( \left(3\,log\left(x + 1\right) -1\right) \, \left(-2\,log\left(x + 1\right) +4\right) = 0 \)

    infatti effettuando la sostituzione \[ \color{red}{}A\color{black}{} = \color{red}{}log\left(x + 1\right) \] otteniamo l'equazione \[ \begin{align*} \left(3\,\color{red}{}log\left(x + 1\right)\color{black}{} -1\right) \,& \left(-2\,\color{red}{}log\left(x + 1\right)\color{black}{} +4\right) = 0 \\\\ &\Downarrow \\\\ \left(3\color{red}{}A\color{black}{} -1\right) \,& \left(-2\color{red}{}A\color{black}{} +4\right) = 0 \end{align*} \]


  • \( \dfrac{1}{log\left(3x^2 -2\right)} + 4\,log\left(3x^2 -2\right) = 0 \)

    infatti effettuando la sostituzione \[ \color{red}{}A\color{black}{} = \color{red}{}log\left(3x^2 - 2\right) \] otteniamo un'equazione di secondo grado \[ \begin{align*} \dfrac{1}{\color{red}{}log\left(3x^2 -2\right)\color{black}{}} + 4\,&\color{red}{}log\left(3x^2 -2\right)\color{black}{} = 0 \\\\ &\Downarrow \\\\ \dfrac{1}{\color{red}{}A\color{black}{}} + 4&\color{red}{}A\color{black}{} = 0 \end{align*} \]


Nel prossimo esempio vediamo la strategia da mettere in campo per risolvere un'equazione di questo tipo.


Esempio


In generale possiamo dire che;

Per risolvere un'equazione del tipo "sostituzione":


  • si effettuano le condizioni di esistenza;

  • si effettua una sostituzione \[ \color{red}{}A\color{black}{} = \color{red}{}log\left(p(x)\right) \] e si risolve l'equazione ottenuta nell'incognita \(\color{red}{A}\)

  • Per ogni soluzione \[ A = s \] risostituiamo \(\color{red}{A}\) con il \(\color{red}{}log\left(p(x)\right)\) e risolviamo l'equazione nell'incognita \(x\) \[ \begin{align*} \color{red}{}A\color{black}{} &= s \\\\ &\Downarrow \\\\ \color{red}{}log\left(p(x)\right)\color{black}{} &= s \end{align*} \]
