3
Limiti di funzioni elementari

/




Funzione \(\boldsymbol{y = x^{2n}}\) (con \(n \in \mathbb{N}\))


Il dominio di una funzione con esponente pari (quindi della forma \(2n\), con \(n \in \mathbb{N}\)) è \[ D = (-\infty\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} x^{2n} = l^{2n} \\\\ &\lim_{x \to +\infty} x^{2n} = +\infty \\\\ &\lim_{x \to -\infty} x^{2n} = +\infty \end{align*} \]






Funzione \(\boldsymbol{y = x^{2n +1}}\) (con \(n \in \mathbb{N}\))


Il dominio di una funzione con esponente dispari (quindi della forma \(2n +1\), con \(n \in \mathbb{N}\)) è \[ D = (-\infty\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} x^{2n + 1} = l^{2n + 1} \\\\ &\lim_{x \to +\infty} x^{2n + 1} = +\infty \\\\ &\lim_{x \to -\infty} x^{2n + 1} = -\infty \end{align*} \]






Funzione \(\boldsymbol{y = b^{x}}\)


Se \(\boldsymbol{b \gt 1}\)

Il dominio di una funzione esponenziale crescente (quindi con base \(b \gt 1\)) è \[ D = (-\infty\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} b^{x} = b^{l} \\\\ &\lim_{x \to +\infty} b^{x} = +\infty \\\\ &\lim_{x \to -\infty} b^{x} = 0 \end{align*} \]





Se \(\boldsymbol{0 \lt b \lt 1}\)

Il dominio di una funzione esponenziale decrescente (quindi con base \(0 \lt b \lt 1\)) è \[ D = (-\infty\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} b^{x} = b^{l} \\\\ &\lim_{x \to +\infty} b^{x} = 0 \\\\ &\lim_{x \to -\infty} b^{x} = +\infty \end{align*} \]




Funzione \(\boldsymbol{y = log_{b}(x)}\)


Se \(\boldsymbol{b \gt 1}\)

Il dominio di una funzione logaritmica crescente (quindi con base \(b \gt 1\)) è \[ D = (0\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} log_{b}(x) = log_{b}(l) \\\\ &\lim_{x \to 0^{+}} log_{b}(x) = -\infty \\\\ &\lim_{x \to +\infty} log_{b}(x) = +\infty \end{align*} \]





Se \(\boldsymbol{0 \lt b \lt 1}\)

Il dominio di una funzione logaritmica decrescente (quindi con base \(0 \lt b \lt 1\)) è \[ D = (0\,;\,\,+\infty) \] Valgono i seguenti limiti:

\[ \begin{align*} &\lim_{x \to l} log_{b}(x) = log_{b}(l) \\\\ &\lim_{x \to 0^{+}} log_{b}(x) = +\infty \\\\ &\lim_{x \to +\infty} log_{b}(x) = -\infty \end{align*} \]
