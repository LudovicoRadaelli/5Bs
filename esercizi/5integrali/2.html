Edizione ...integrale!

Rispolveriamo la strategia che abbiamo messo in atto per riuscire a derivare una qualsiasi funzione

Primo passo
Abbiamo trovato attraverso la definizione le derivate delle funzioni più comuni

Derivate fondamentali

\(\left(x^{\alpha}\right)' = \alpha\,x^{\alpha - 1}\,\,\,\) con \(\alpha \in \mathbb{R}\,\,\,\,\,\)
\(\left(ln(x)\right)' = \dfrac{1}{x}\)
\(\left(sin(x)\right)' = cos(x)\)
\(\left(cos(x)\right)' = -sin(x)\,\,\,\,\)
\(\left(e^x\right)' = e^x\,\,\,\,\)
\( \begin{align*} \left(tan(x)\right)' &= \dfrac{1}{cos^2(x)} \\\\ &= 1 + tan^2(x) \end{align*} \)
\(\left(arcsin(x)\right)' = \dfrac{1}{\sqrt{1 -x^2}}\,\,\,\,\,\)
\(\left(arccos(x)\right)' = -\dfrac{1}{\sqrt{1 -x^2}}\,\,\,\,\,\)
\(\left(arctan(x)\right)' = \dfrac{1}{1 +x^2}\,\,\,\,\,\)
\(\left(arccot(x)\right)' = -\dfrac{1}{1 +x^2}\,\,\,\,\,\)


Secondo passo
Abbiamo individuato una manciata di regole per derivare funzioni più complesse a partire da quelle della lista.

Regole di derivazione

Derivata del prodotto di una costante per una funzione
\(\left(c\,f(x)\right)' = c\,f'(x)\)

Derivata delle somma di due funzioni
\(\left(f(x) + g(x)\right)' = f'(x) + g'(x)\)

Regola di Leibniz
\(\left(f(x) \cdot g(x)\right)' = f'(x)\,g(x) + f(x)\,g'(x)\)

Derivata del rapporto di due funzioni
\(\left(\dfrac{f(x)}{g(x)}\right)' = \dfrac{f'(x)\,g(x) - f(x)\,g'(x)}{\big[g(x)\big]^2}\)

Derivata di una funzione composta

\(\big[f\big(g(x)\big)\big]' = f'\big(g(x)\big)\,g'(x)\)

Derivata dell'inversa di una funzione
\(\big[f^{-1}(x)\big]' = \dfrac{1}{f'(f^{-1}(x))}\)

Adottiamo la stessa strategia per il calcolo degli integrali.

Dalla lista delle derivate delle funzioni più comuni otteniamo la seguente lista degli integrali delle funzioni più comuni.

Integrali fondamentali

\[ \int x^{\alpha} \,\,dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]
\[ \int \dfrac{1}{x} \,\, dx = ln\left(x\right) + c \quad \text{con} \quad x \gt 0 \]
\[ \int e^{x} \,\, dx = e^{x} + c \]
\[ \int sin(x) \,\, dx = -cos(x) + c \]
\[ \int cos(x) \,\, dx = sin(x) + c \]
\[ \int \dfrac{1}{cos^2(x)} \,\, dx = tan(x) + c \]
\[ \int \dfrac{1}{sin^2(x)} \,\, dx = -cotan(x) + c \]
\[ \int \dfrac{1}{\sqrt{1 -x^2}} \,\, dx = arcsin(x) + c \]
\[ \int \dfrac{1}{1 + x^2} \,\, dx = arctan(x) + c \]


In maniera altrettanto immediata deduciamo la linearità dell'integrale a partire da quella della derivata.

Regole di integrazione

Linearità dell'integrale - 1
\[ \int \big[f(x) + g(x)\big] \,\,dx = \int f(x) \,\,dx + \int g(x) \,\,dx \] É possibile distribuire l'integrale su una somma di funzioni.
Linearità dell'integrale - 2
\[ \int \big[a \,f(x) \big] \,\,dx = a \, \int f(x) \,\,dx \] É possibile portare le costanti moltiplicative fuori dal segno di integrazione.
