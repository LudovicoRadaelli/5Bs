Integrali fondamentali

\[ \int x^{\alpha} \,\,dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]
\[ \int \dfrac{1}{x} \,\, dx = ln\left(\left|x\right|\right) + c \quad \text{con} \quad x \gt 0 \]
\[ \int e^{x} \,\, dx = e^{x} + c \]
\[ \int a^{x} \,\, dx = \dfrac{a^{x}}{ln(a)} + c \]
\[ \int sin(x) \,\, dx = -cos(x) + c \]
\[ \int cos(x) \,\, dx = sin(x) + c \]
\[ \int \dfrac{1}{cos^2(x)} \,\, dx = tan(x) + c \]
\[ \int \dfrac{1}{sin^2(x)} \,\, dx = -cotan(x) + c \]
\[ \int \dfrac{1}{\sqrt{1 -x^2}} \,\, dx = arcsin(x) + c \]
\[ \int \dfrac{1}{1 + x^2} \,\, dx = arctan(x) + c \]


Regole di integrazione

Linearità dell'integrale - 1
\[ \int \big[f(x) + g(x)\big] \,\,dx = \int f(x) \,\,dx + \int g(x) \,\,dx \] É possibile distribuire l'integrale su una somma di funzioni.
Linearità dell'integrale - 2
\[ \int \big[a \,f(x) \big] \,\,dx = a \, \int f(x) \,\,dx \] É possibile portare le costanti moltiplicative fuori dal segno di integrazione.
