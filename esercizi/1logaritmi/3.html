Esercizio 1


\[ 2\,log\left(x + 2\right) - log\left(-2x + 10\right) = 0 \]


Soluzione: \(\,\,\,x = \sqrt{15} - 3\)





Esercizio 2


\[ log\left(4x + 12\right) - 2\,log\left(2\right) = log\left(5\right) + log\left(x +2\right) \]


Soluzione: \(\,\,\,x = -\dfrac{7}{4}\)





Esercizio 3


\[ 2ln\left(3x -2\right) + \dfrac{1}{ln\left(3x -2\right)} = 3 \]


Soluzione:
\(x = \dfrac{2 + \sqrt{e}}{3} \quad \text{oppure} \quad x = \dfrac{2 + e}{3}\)
